## 📌 Project Overview

This project presents an interactive **IPL (Indian Premier League) Data Analysis Dashboard** built using Microsoft Excel.
The dashboard provides insights into team performance, match outcomes, player achievements, and key trends from IPL seasons (2008–2025).

The goal of this project is to demonstrate data analysis, visualization, and storytelling skills using Excel.

---

## 🎯 Objectives

* Analyze IPL match data across multiple seasons
* Identify winning patterns and team performance trends
* Understand the impact of toss decisions on match results
* Highlight top-performing players (Man of the Match)
* Create an interactive and user-friendly dashboard

---

## 🛠️ Tools & Technologies Used

* **Microsoft Excel**

  * Pivot Tables
  * Pivot Charts
  * Slicers & Filters
  * Data Cleaning Techniques
* Data Visualization Techniques

---

## 📊 Dashboard Features

* 📅 **Year-wise Analysis**
  Filter data based on IPL seasons

* 🏆 **Winning Team & Runner-up**
  Displays champions and finalists for selected years

* 🎖️ **Man of the Match Insights**
  Identifies top-performing players

* 🪙 **Toss Decision Analysis**
  Shows how toss decisions affect match outcomes

* 📍 **Venue Analysis**
  Highlights top venues based on matches played

* 📈 **Team Performance Metrics**
  Visual comparison of team wins and trophies
